# SEO Plan

## Primary keyword clusters

- `hourly rate calculator`
- `day rate calculator`
- `retainer calculator`
- `hourly vs project pricing`
- `freelance pricing calculator`

## Content model

- 工具页负责承接高意图搜索
- 指南页负责解释收费模型与争取长尾词
- 每个新页面都要有明确的主关键词和内链位置

## Expansion directions

- `consulting rate calculator`
- `freelance project quote calculator`
- `monthly retainer pricing guide`
- `how to price strategy workshops`
- `proposal template for freelancers`

## Rules

- 每个页面只聚焦一个主要问题
- 标题直接匹配用户搜索意图
- 文案尽量简单，避免术语堆叠
- 工具页和指南页互相内链
