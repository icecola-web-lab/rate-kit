# Launch Checklist

## Product

- 检查首页和 3 个计算器的核心文案
- 确认 4 种语言都能正常切换
- 手动验证主要输入输出逻辑

## Technical

- 配置 `NEXT_PUBLIC_SITE_URL`
- 如需统计，配置 `NEXT_PUBLIC_PLAUSIBLE_DOMAIN`
- 执行 `pnpm typecheck`
- 执行 `pnpm lint`
- 执行 `pnpm build`

## SEO

- 绑定正式域名
- 在 `Google Search Console` 提交 `sitemap.xml`
- 检查 `robots.txt` 与 canonical
- 确认首页和指南页的分享图正常

## Distribution

- 在 `X`、`Reddit`、`Indie Hackers`、相关论坛发布首轮介绍
- 为日语和中文社区分别准备简短介绍帖
- 记录最先有点击和收录的页面

## Post-launch

- 观察用户更常使用哪一个计算器
- 优先扩展搜索量更高的长尾报价页面
- 每周补充 1 到 2 篇解释型 SEO 页面
