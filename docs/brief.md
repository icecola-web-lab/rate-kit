# Rate Kit Brief

## Positioning

Rate Kit 是一个面向自由工作者、顾问与小型工作室的多语言报价工具站，目标是在无需登录和无需后端依赖的前提下，快速完成与服务报价相关的高频计算。

## Core value

- 用最短路径完成 `hourly rate`、`day rate`、`retainer` 计算
- 通过 SEO 获取全球自然流量
- 用多语言覆盖英语、日语、繁体中文、简体中文用户
- 保持产品轻量，降低维护成本与上线门槛

## MVP scope

- 首页落地页
- 3 个核心计算器页面
- 1 个关于收费模型的指南页
- 隐私政策与条款页面
- sitemap、robots、Open Graph、多语言路由

## Success signals

- 站点可稳定部署到 `Vercel`
- 每个页面具备独立 metadata、FAQ 与结构化数据
- 用户无需注册即可完成主要任务
- 后续可以扩展更多长尾报价与模板型页面
