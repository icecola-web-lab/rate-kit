# Monetization Plan

## Phase 1

首发阶段不接付费，目标是先用免费报价工具页获得搜索流量和真实需求反馈。

## Phase 2

在不增加复杂后端的前提下，优先考虑以下变现方式：

- 数字产品：销售提案模板、报价模板、定价表、Scope 清单
- 联盟分发：推荐发票、时间追踪、提案、客户管理相关 SaaS
- 广告或赞助：当自然流量稳定后再接入

## Phase 3

如果验证到高价值需求，再考虑轻量付费功能：

- 保存多个报价方案
- 导出报价 PDF
- 客户可分享的报价链接
- 不同收费模型的组合比较

## Why this order

- 先做流量，再做收费，决策风险更低
- 工具站天然适合 SEO 和数字产品分发
- 没有自建服务器时，先避免账户和权限复杂度
